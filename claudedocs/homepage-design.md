## 2. Feature Overview (6가지 핵심 기능 카드)

![Dashboard Preview & Feature Cards](./actual-dashboard-preview.png)

### 구성 요소
6개 번호형 카드 레이아웃 (가로 스크롤)

| 번호 | 기능 | 설명 |
|------|------|------|
| 1 | 랜딩페이지 생성 | 누구나 5분만에 쉽게 제작. 홈페이지 및 DB수집용 랜딩페이지 제작 가능. |
| 2 | 실시간 DB 수집 및 관리 | 담당자별 DB 관리. 콜센터에서 활용 가능한 기능 추가. |
| 3 | DB예약 스케줄 관리 | 캘린더로 관리하는 DB 스케줄. 예약, 방문 스케줄 별도 관리 가능. |
| 4 | 트래픽 분석 | UTM, 기기별 유입경로 분석. 유입경로별 전환을 확인. |
| 5 | 부서별/담당자별 성과 분석 | 콜 전환 성과 분석. 월별·개인별 요약 리포트 제공. |
| 6 | 광고 픽셀 & API 연동 | 광고 진행을 위한 연동 가이드 제공. 효율 및 전환 체크 가능. |

### 섹션 제목
- **강력한 기능** (레이블)
- **비즈니스 성장에 필요한 모든 것** (헤드라인)
- "랜딩페이지 제작부터 DB 관리, 분석까지 하나의 플랫폼에서 해결하세요" (서브텍스트)

![Landing Page Feature](./actual-industry-section.png)

**헤드라인**: "개발자 없이 누구나 5분만에 랜딩페이지 완성"

**기능 목록** (아이콘 카드):

![Landing Page Details](./actual-feature-showcase1.png)

| 기능 | 설명 |
|------|------|
| 이미지 or 영상 삽입 | 원하는 이미지와 동영상만 있다면 간단히 랜딩페이지 URL 생성이 가능합니다. |
| DB수집 마감 타이머 | DB 수집 마감 타이머 날짜 설정이 가능합니다. |
| DB 수집 폼 | 이름, 연락처, 단답형/선택형 질문 등을 추가하여 DB 수집이 가능합니다. |
| 상담신청 버튼 | 상담신청 버튼 추가/삭제로 랜딩페이지 혹은 홈페이지로 활용이 가능합니다. |
| 전화연결 버튼 | 전화연결 버튼을 추가 및 삭제할 수 있습니다. |
| 신청완료 이미지 | 신청 완료 후 노출할 이미지를 설정할 수 있습니다. |
| 완료 메시지 (텍스트) | 신청 완료 후 표시될 안내 메시지를 자유롭게 작성할 수 있습니다. |
| 안내 멘트 (텍스트) | 랜딩페이지 하단에 표시할 안내 문구를 설정할 수 있습니다. |

### 4-2. 실시간 DB 수집 및 관리 상세

![DB Management](./actual-feature-showcase2.png)

**헤드라인**: "(모든 DB를) 한 곳에서 통합 관리"

**설명**: "실시간으로 수집된 DB를 간편하게 관리하실 수 있습니다. 콜센터 운영 중인 경우, DB배분 기능으로 콜 관리를 할 수 있으며, 퍼널리 외 매체의 DB도 수동추가로 통합 관리가 가능합니다."

**DB 현황 미리보기**:
- DB 현황 테이블 (이름, 연락처, 상태, 결과)
- 샘플 데이터: 상담전 / 진행중 / 예약확정 / 거절 상태 표시
- 통계: 총 DB 128 / 진행중 43 (33.6%) / 확정 21 (16.4%)

**기능 목록**:

| 기능 | 설명 |
|------|------|
| DB 현황 | 랜딩페이지로 수집된 DB 리스트를 확인할 수 있습니다. 수집된 DB 항목, 콜 담당자, 상담 담당자, 콜 결과 등 확인이 가능합니다. |
| DB 배분 | 수집된 DB를 콜 담당자에게 임의로 배분합니다. |
| DB 수동 추가 | 퍼널리 이외의 방법으로 수집된 DB는 수동추가 기능 이용이 가능합니다. |
| 결과 처리 | 상담 결과를 선택하여 스케줄을 관리할 수 있습니다. |

### 4-3. 트래픽 분석 & DB 리포트 상세

![Traffic Analysis](./actual-feature-showcase3.png)

**설명**: "DB 유입 경로를 PC·모바일·태블릿으로 구별하여 분석합니다. 트래픽 유입경로뿐만 아니라 전환된 경로까지 체크하고, 결일·결일별 DB 결과를 한눈에 확인할 수 있습니다."

**유입 경로 분석 차트**:
- PC: 58%
- Mobile: 35%
- Tablet: 7%

**데일리 DB 리포트**:
- DB 유입: 128 / 예약확정: 21 / 전환율: 16.4%

**기능 목록**:

| 기능 | 설명 |
|------|------|
| 트래픽 분석 | DB 유입 경로를 PC, MOBILE, TABLET으로 구별하여 분석합니다. |
| 유입경로 전환 분석 | 트래픽 유입경로뿐만 아니라 전환된 경로까지 체크할 수 있습니다. |
| DB 리포트 | 일별·월별 유입된 DB의 결과를 한눈에 체크할 수 있습니다. |
| 데일리 성과 측정 | DB 유입, 상담전, 거절, 진행중, 예약확정, 결제 횟수, 결제금액 등 데일리 성과 측정이 가능합니다. |
